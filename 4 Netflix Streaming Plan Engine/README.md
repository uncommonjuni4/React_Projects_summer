🎬 Project 4: Netflix Streaming Plan Engine
Description:
A functional subscription sub-billing engine crafted in React JS to perfect boundary gatekeeping. Features a Logical OR (||) short-circuit fallback system to safely output legal profiles when field inputs are empty, and utilizes Logical AND (&&) conditional execution to automatically display operational pricing warnings directly on the client UI.
